# m1-style (サイドプロジェクト)

`html/m1-style.html` は、E5確定後に審査員ごとの旗判定を段階表示する実験用ウィンドウです。

## 仕様（現時点）

- トリガー: `match_snapshots` の最新行が更新されたとき（= E5再実行でも再演出）
- 表示内容: 審査員名 + 旗を上げた側のチーム名（点数は表示しない）
- 取得方式: ポーリングのみ
- 保存: サーバへの追加保存なし

## 使い方

1. `html/m1-style.html` を開く
2. 必要なら `?venue=default` のように会場コードを付ける
3. 管理画面で E5 を実行すると自動で演出が開始する

## 補足

- これは独立画面のため、本体画面（`admin.html` / `obs-scoreboard.html`）への組み込みはしていません。
- 会場モニタ向けに大きめ文字と高コントラストを優先しています。
